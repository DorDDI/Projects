C++ 
program set and queue machine that manage queue principles using C++ and OOP syntax

header - contain 3 files of the prototypes of the program
source - contain 3 files of the prototypes of the program and one main file to start the program

The program memu:
1. Enter set Menu
2. Enter queue Menu
3. Exit

The set Menu:
1. Show set 1
2. Show set 2
3. Add element to set 1
4. Add element tp set 2
5. Show set1 U set 2 (union)
6. Exit

The queue Menu:
1. Show Queue
2. Insert new element
3. Remove element
4. Check the first element
5. Exit
