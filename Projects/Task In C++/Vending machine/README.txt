C++ 
program of a vending machine that sell products, recive money and use a stack managment menu using C++ and OOP syntax

header - contain 6 files of the prototypes of the program
source - contain 6 files of the prototypes of the program and one main file to start the program

The program memu:
1. Enter Fresh juice Shop
2. Enter Integer Stack
3. Exit

The Fresh juice Shop Menu:
1. Orange juice
2. Carrot juice
3. Pomegranate
4. Exit

The Integer Stack Menu:
1. Push new element
2. Pop element
3. Show the first element
4. Check if empty
5. Exit
